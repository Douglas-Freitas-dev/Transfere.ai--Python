Bem-vindo ao meu repositório

Realizar a instalação da última versão do Python 3.9.5
link(https://www.python.org/downloads/)

Primeiro Passo
Verificar se foi instalado corretamente
python --version
Na tela deverá aparecer
Python 3.9.5 
 

Segundo Passo
Instalar a extensão do PySimpleGUI, abra o cmd ou Shell e digite o seguinte comando
pip install pysimplegui
Aguarde a instalação ser concluída e feche o terminal

Terceiro Passo
Configurar o interpretador do python no Visual Studio Code
Ir até a parte de Extensions, pesquisar Python, selecionar a primeira opção e instalar.
 

Configuração finalizada.
